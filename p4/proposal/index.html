Title

Web Content Management System For Small Non-profit Organizations

Description

This CMS will concentrate on small fund racing organizations. It will hold up to 10 empty web pages where the user can add content. User /admin will be able to register, log in, choose a simple color template, upload site banner and logo, and insert text and pictures.
